Form Elements

Tag Atributes Description
<form>
  • action
  • method
 Represents a document section containing interactive controls for submitting information
<input>
  • type
  • name
  • value
  • required
  • id
 Used to create interactive controls for web-based forms in order to accept data from the user
<textarea>
  • name
  • cols
  • rows
  • placeholder
  • id
 Represents a multi-line plain-text editing control
<select>
  • name
  • required
  • size
  • multiple
  • id
 Represents a control that provides a menu of options
<option>
  • value
 Is used to define an item contained in a <select>
<datalist>
  • id (list atribute in the <input>)
 Contains a set of <option> elements that represent the permissible or recommended options available to choose from
<label>
  • for
 Represents a caption for an item in a user interface

type values

Atribute Description
checkbox A check box allowing single values to be selected/deselected
date A control for entering a date (year, month, and day, with no time)
number A control for entering a number
password A single-line text field whose value is obscured
radio A radio button, allowing a single value to be selected out of multiple choices
submit A button that submits the form
text A single-line text field
